# Retail-Sales-Analysis
<p>The project involves setting up a retail sales database, performing exploratory data analysis (EDA), and answering specific business questions through SQL queries.</p>
<b>OBJECTIVES</b>
<hr>
<p>1. <b>Set up a retail sales database:</b> Create and populate a retail sales database with the provided sales data.</p>
<p>2. <b>Data Cleaning:</b> Identify and remove any records with missing or null values.</p>
<p>3. <b>Exploratory Data Analysis (EDA):</b> Perform basic exploratory data analysis to understand the dataset.</p>
<p>4. <b>Business Analysis:</b> Use SQL to answer specific business questions and derive insights from the sales data.</p>

<b>FINDINGS</b>
<hr>
<p><b>Customer Demographics:</b> The dataset includes customers from various age groups, with sales distributed across different categories such as Clothing and Beauty.</p>
<p><b>High-Value Transactions:</b> Several transactions had a total sale amount greater than 1000, indicating premium purchases.</p>
<p><b>Sales Trends:</b> Monthly analysis shows variations in sales, helping identify peak seasons.</p>
<p><b>Customer Insights:</b> The analysis identifies the top-spending customers and the most popular product categories.</p>
